# BerTiga Compro

Website company profile + katalog template digital dengan cart statis.

## Cara menjalankan

```bash
npm install
npm run dev
```

Buka:

```txt
http://localhost:3000
```

## File penting

- `app/page.tsx` halaman utama + cart statis
- `app/globals.css` semua styling dan animasi
- `public/BerTiga_Logo_Icon.png` logo BerTiga
